Inspect Element Menu

This is a custom inspect element menu that adds a draggable panel with various tools and effects to any webpage. It includes fun animations, page information tools, editing capabilities, and more.

How to Use

To use this on any website:

  1. Copy the JavaScript code from inspect-element.js
  2. Create a bookmark in your browser
  3. Paste the code as the bookmark URL (javascript:...)
  4. Click the bookmark on any page to activate the menu

Features

  • Executables: Fun effects like Matrix rain, snow, page shake, emoji storm
  • Info: View cookies, localStorage, images, links
  • Page Edit: Edit text, invert colors, mirror page, crazy fonts
  • Network: Ping test, IP info
  • Console: Run JavaScript expressions
  • Tweaks: Hide images/ads, grayscale, no animations, dark mode, font size/zoom sliders
